## Decisions

1. **Distroless final image** (`gcr.io/distroless/static-debian12:nonroot`): Minimal CVE surface, no shell or package manager, nonroot UID 65532 pre-configured.
2. **Deploy manifests in `deploy/`**: All Kubernetes manifests in one flat directory, applied in order (namespace → serviceaccount → configmap → deployment). No Helm.
3. **Workload Identity env vars omitted from Deployment**: Owned by AKS webhook. Manual setting would create maintenance hazard.
4. **Placeholder pattern with `# TODO:`**: Values from Bishop's Azure setup referenced in `docs/azure-setup.md`.
5. **`readOnlyRootFilesystem: true`**: Container writes only to remote Blob Storage; local filesystem is read-only.
